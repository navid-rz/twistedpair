# twistedpair — Documentation

This `docs/` folder contains high-level documentation for the `twistedpair` project: a Python-based simulator for high-speed SerDes and system modeling.

Files:
- `overview.md` — Project purpose and scope
- `installation.md` — How to install and run examples
- `architecture.md` — High-level architecture and data flow
- `modules.md` — Summary of important modules and responsibilities
- `usage.md` — How to run demos and common workflows
- `examples.md` — Example scripts and quick commands
- `configuration.md` — Config files and presets
- `api_reference.md` — Where to find code-level APIs
- `contributing.md` — How to contribute, run tests, and coding notes

Start with `overview.md` to get an orientation of the project.
